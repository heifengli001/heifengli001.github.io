股票的几个指标含义

指标释义：

  • MA: Moving Average 移动平均线
  • EMA： Exponent Moving Average 指数移动均线
  • SMA: Simple Moving Average 简单移动均线
  • MACD：Moving Average Convergence and Divergence 指数平滑移动移动平均线
    • Convergence and Divergence 收敛和发散
  • KD: Stochastics oscillator 随机震荡指标
  • WMA: Weighted Moving Average 加权移动平均线
  • ATR: Average True Range 平均真实波动幅度
  • RSI : Relative Strength Index 相对强弱指

MA （均线）

Moving Average

计算规则： 近N根K线的收盘价/周期N = N周期均线
特点：均线是历史价格统计，具有滞后性
应用：追踪趋势，观察支撑点

  • 单均线： 短中周期均线，提供买卖点。
    • 葛南维八大法则

1、移动平均线从下降逐渐走平且略向上方抬头，而股价从移动平均线下方向上方突破，为买进信号。
2、股价位于移动平均线之上运行，回档时未跌破移动平均线后又再度上升时为买进时机。
3、股价位于移动平均线之上运行，回档时跌破移动平均线，但长期移动平均线继续呈上升趋势，此时为买进时机。
4、股价位于移动平均线以下运行，突然暴跌，距离移动平均线太远，极有可能向移动平均线靠近（物极必反，下跌反弹），此时为买进时机。
5、股价位于移动平均线之上运行，连续数日大涨，离移动平均线愈来愈远，说明近期内购买股票者获利丰厚，随时都会产生获利回吐的卖压，应暂时卖出持股。
6、移动平均线从上升逐渐走平，而股价从移动平均线上方向下跌破移动平均线时说明卖压渐重，应卖出所持股票。
7、股价位于移动平均线下方运行，反弹时未突破移动平均线，且移动平均线跌势减缓，趋于水平后又出现下跌趋势，此时为卖出时机。
8、股价反弹后在移动平均线上方徘徊，而移动平均线却继续下跌，宜卖出所持股票。

  • 双根均线：短/中周期均线
    • 短周期突破长周期上涨金叉 ，买入信号
    • 短周期下穿长周期下跌死叉，卖出信号
  • 多根均线：短中长周期均线，判断趋势

MACD （指数平滑移动移动平均线 ）

Moving Average Convergence and Divergence

双线MACD

默认参数：12,26,9
DIF:快线 = 12EMA-26EMA
DEA:慢线 = 快线的移动平均线（周期为9）
能量柱：n*（快线-慢线）
应用：

  • 根据快线DIF判断
    • 快线在0轴上方为多头趋势
    • 快线在0轴下方为空头趋势
  • 根据快线DIF和DEA的交叉判断多空信号
    • 快线向上突破慢线金叉，买入信号
    • 快线向下穿过慢线死叉，卖出信号
  • 结合趋势+交叉
    • 快线在0轴上方，并金叉则为 多头信号
    • 快线在0轴下方，并死叉则为 空头信号
  • 指标和价格背离，用来判断行情反转
    • 如 价格创新高但是MACD的趋势向下

KD (随机震荡指标)

Stochastics Ooscillator

范围：0-100
默认参数：9,3,3
计算规则：

  1. 计算RSV = （当前收盘价-近9根K线最低价）/ (近9根K线最高价-近9根K线最低价) * 100
  2. 计算K值 = 2/3前一天的K值 + 1/3当天的RSV (即表示 RSV的周期为3的加权移动平均线)
  3. 计算D值 = 2/3前一天的D值 + 1/3当天的K值 (即表示 K值的周期为3的加权移动平均线)

对当前收盘价极其灵敏
应用：当做入场信号

  • 到达20轴，并上穿20轴形成金叉 ，入场做多
  • 下跌80轴，并跌破90轴行程死叉，离场做空

配合MACD一起使用

ATR (平均真实波动幅度)

Average True Range

计算规则：
1.计算TR = MAX(最高价-最低价（振幅），最高价-前收盘价（涨幅），前收盘价-最低价（跌幅）)
2.计算ATR = 近期N根K线的TR数值求和再平均
应用：制定开仓数量和止损未知

  • 海龟交易法：
    • 计算开仓数量 （账户总资金 * 1%）/ (ATR * 合约每点点值)
        1. 计算价值波动量DV = ATR * 合约每点点值
      • 2.计算单笔止损金额 = 账户总资金 * 1%
      • 3.计算开仓数量 = 单笔止损金额/DV
    • 指定止损/加仓规则
      • 止损=开仓价格-（2*ATR）
      • 加仓= 盈利突破 1/2*ATR时加仓，最多4个单位，每次加仓后 止损位置提高1/2倍ATR

RSI （相对强弱指标）

Relative Strength Index

默认参数：14
取值：0~100

  • 0~30 极弱 超卖区
  • 30~50 弱
  • 50~70 强
  • 70~100 极强 超买区

计算规则
A = 近14根K线阳线的涨幅之和
B= 近14根K线阴线的跌幅之和的绝对值
RSI= A/(A+B) * 100 = 上涨的波动/总波动
应用：

  • 入场离场信号
    • RSI在超卖区(30轴)向上突破30轴，做多信号。（情况为最近连续下跌，出现大阳或者连续阳线）
    • RSI在超买区(70轴)向下刺穿70轴，做空信号。（情况为最近连续上涨，出现大阴或者连续阴线）
  • RSI指标背离
    • 判断行情反转，价格上涨斜率向上，但是RSI峰值斜率向下。（情况为涨幅降低，上涨节奏减缓）

BOLL（布林线）

Bollinger Bands

默认参数：20
组成：

  • 中轨线：N日移动平均线
  • 下轨线：中轨线+2倍标准差
  • 下轨线：中轨线-2倍标准差
    • 标准差为方差的平方根，表示离散程度
    • 利用标准差可以通过正态分布得知，可以覆盖95%的样本数据

应用：

  • 支撑和阻力
    • 一般认为 布林的上轨线有阻力作用，下轨线有支撑作用
  • 超买超卖
    • 上涨超过上轨为超买
    • 下跌超过下轨为超卖
  • 判断趋势
    • 根据布林线的趋势可以判断多空趋势

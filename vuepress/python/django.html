# Django 学习笔记

# 准备工作

环境 Python Pip django

  • 安装Django

    使用pip 进行安装 命令 pip install Django==2.2

# 创建Django项目

  • 在目标文件夹执行命令 jango-admin startproject ProjectName

    没有报错表示执行成功

  • 项目结构分析

--ProjectName
   manage.py           //项目入口（项目管理器
   ProjectName         //项目容器目录 （不建议修改名称
       __init__.py     //
       settings.py     //  配置文件
       urls.py         // url 配置
       wsgi.py         // Python 服务器的网关接口

# 基础知识

n, a, b = 0, 0, 1
while n < max:

  1. reduce

接收两个参数（方法，队列），将方法==必须接收两个参数==并应用在方法结果以及下一元素上,返回累积结果

    // 使用方法
    new_result = reduce(function(),Iterable)
    // 如：
    reduce(f, [x1, x2, x3, x4]) = f(f(f(x1, x2), x3), x4)
    
    // eg:
    from functools import reduce
    def add(x, y):
        return x + y
    
    a = reduce(add, [1, 3, 5, 7, 9])
    
    print(a)
    
    // 程序输出结果：
    >>> 25
